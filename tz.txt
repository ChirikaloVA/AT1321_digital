Принципы устройства графического интерфейса СПРД Атомтех
В настоящей записке приведена максимальная программа, которую можно выполнять
поэтапно, если прибор будет пользоваться успехом на рынке. Рассматриваются следующие этапы:
a)	Прибор в базовой комплектации с БД 1X1" NaI
b)	Для оценки высокой мощности дозы добавить счетчик Гейгера-Мюллера.
Это необходимо если прибор используется для измерения мощности дозы в чрезвычайных ситуациях.
Для этого необходимо ввести второй порог срабатывания сигнализации, обладающий высокой надежностью.
c)	Добавить детектор нейтронного излучения He-3 (если есть спрос)
d)	Добавить GPS и небольшую камеру (как в мобильных телефонах), встроенное устройство считывания данных
(как в системе, которая укладывается в рюкзак) и устройство перехвата
e)	Одновременно разработать сцинтилляционный детектор с лучшим разрешением (LaCl-3, LaBr-3).
Примечание: Кроме типового названия прибора, желательно дать более интуитивно понятное название,
например: RadSearcher, RadSeeker
1	Средства навигации
Под экраном имеется 3 кнопки. В упрощенных режимах они называются 1, 2, 3.
В режиме идентификации радионуклидов вручную (=расширенный режим) кнопки называются <----, *, ----->.
В этом случае две стрелки двигают подсветку по различным пунктам, а нажатие кнопки * означает
принятие выбора (подсвеченного пункта), что равнозначно нажатию клавиши enter. Такой способ необходим для
эффективной работы с многочисленными меню режима ручной идентификации радионуклидов.
Во всех других случаях используется упрощенный принцип функциональных кнопок.
Над кнопками 1,2,3 всегда высвечивается 3 подсказки. Правая кнопка всегда Next (След.) или Back (Назад).
Кнопка Next переключает на следующий (по порядку) экран, кнопка Back возвращает в режим СПРД
(который используется чаще всего и является начальным пунктом всех меню).
Если кнопки 1,2,3 используются (во всех режимах кроме RID), команда, указанная над кнопкой,
вызывается нажатием этой кнопки. Таким образом, команды вызываются намного быстрее по сравнению
с режимом RID (там сначала нужно выделить необходимую команду, а затем нажать * (enter)).
(наш опыт работы с СПРД Sigma показал, что этот метод намного медленнее и вызывает затруднения у пользователей
в стрессовой ситуации). Некоторые кнопки получают другое назначение в зависимости от состояния прибора.
Например, для кнопки Start/Stop сначала показывается команда Start. После нажатия команда меняется на Stop.
Это позволяет уместить в окне больше команд, что снижает количество окон.
Прибор имеет следующие режимы работы:
Пуск и инициализация (Init); сюда также входит время прогрева, необходимое для повышения надежности
перекалибровки энергетической шкалы. Если прибор включается только для проведения технических работ,
прогрев можно попустить (также можно пропустить и последующую перекалибровку).
Выводится важная информация…

Следующий режим - Перекалибровка (Cal) энергетической шкалы.
Наш опыт работы с различными приборами показывает, что проведение перекалибровки настоятельно рекомендуется в
начале работы (она также может понадобиться перед проведением тонких измерений по юридическим причинам).
По ее завершении выводится информация о качестве результатов перекалибровки. Если критические параметры
(например, центр пика и ПШПМ - полная ширина на половине высоты) выходят за пределы диапазона, можно провести
повторную калибровку, иначе необходим ремонт, так как радионуклид. В полевых условиях используется источник KCl
небольшого объема, время набора как минимум 300 с. В лабораторных условиях используется источник Cs-137.
Предлагается сделать линеаризацию калибровочной кривой по энергии с полиномом более высокого порядка,
чтобы отклонения … измерялись при помощи … В процессе перекалибровки регулируется только крутизна,
другие параметры полинома не регулируются (если требуется, должно выполняться только специально обученными
техниками).
Затем пользователь может выбрать один из следующих режимов работы:
1.1	SPRD
RID
Таймер/счетчик (T/C)
MCS и
Setup (Настройки)

Режим SPRD является режимом по умолчанию. Он сочетает в себе функции ПРД (грубый счет и поисковую и
пороговую сигнализацию) и автоматическую идентификацию (RID), которая запускается при выполнении условий
срабатывания сигнализации.  По истечении некоторого времени делаются попытки автоматической идентификации,
результат выводится на экран. Во время набора спектра, поисковые функции как в PRD должны быть активны.
Поскольку важно, чтобы результат идентификации быстро отображался на экране после срабатывания сигнализации,
обработка должна проводиться одновременно со сбором данных. Если это не возможно, передача спектра и попытка
идентификации должны производиться через каждые 1000 импульсов. Время обработки должно быть максимально коротким.

Режим RID похож на любую идентификацию радионуклидов. Измерение гамма-радиации начинается вручную.
После установленного времени набор останавливается и оценивается гамма-спектр.
Идентифицированные радионуклиды показываются на экране. Файл спектра можно сохранить вместе с результатами идентификации.

Режим Таймер/счетчик (T/C) позволяет запустить грубое измерение гамма и нейтронного излучения.
После остановки на экран выводятся средняя скорость счета и ее абсолютная и относительная погрешности.
Этот режим полезен если нужно распознать слабый источник гамма или нейтронного излучения.
Можно провести измерение суммарной дозы за определенный период.
Измерение дозы может использоваться для оценки гамма-облучения оператора за определенный период времени.

Режим MCS похож на режим СПРД. Тем не менее, при срабатывании сигнализации автоматический набор спектра
не производится. Полученные кривые MCS можно сохранить в файл (автоматическая нумерация файлов).

В каждом режиме есть меню Setup (Настройки). Самые важные параметры показаны в окне Setup Summary.
Это позволяет избежать навигации по многочисленным меню, если необходимо проверить определенную группу параметров.
В различных режимах имеются окна Настроек. Некоторые часто требующиеся функции (например, отключить звук),
доступны прямо в окне соответствующего режима… Однако все окна настроек открываются только при вводе пароля
(минимум один раз). Редко используемые тонкие настройки доступны только через соединение с компьютером.
Специальные возможности
Автоматический режим
При подключении к ПК должна быть возможность управлять автоматическим многократным измерением спектра,
в такой последовательности: Старт, измерение в течение установленного времени, оценка (идентификация изотопа),
передача спектра на ПК и сохранение в файл с автоматической возрастающей нумерацией.
Этот режим очень полезен при проведении многократных измерения с идентификацией радионуклидов для проверки
надежности результатов. Среди прочего, это очень удобно для проверки доверительной вероятности результатов идентификации.
Стабилизация:
Светодиодная измерительного тракта, температурный датчик на кристалле для достижения хорошего температурного равновесия.
Использовать изолирующий слой (амортизация и тепловая изоляция) вокруг кристалла, кристалл поместить внизу.
2	Испытания детекторов
Проверяется скорость счета NaI детектора. Для успешной проверки она должна быть > 0. Слишком высокая скорость
счета проверяется по мощности дозы. Мощность дозы может быть показана только при наличии спектра.
Из нашего опыта, нормальная фоновая мощность дозы <  0,3 мкЗв/ч. При обнаружении более высокой мощности дозы
выводится сообщение: Слишком высокая мощность дозы, перейдите в место с более низким фоном.
Следует разработать похожие критерии для других детекторов.

3	Настройка через ПК
Должна быть возможность сохранить все настройки прибора в компьютерном файле и настраивать прибор,
загрузив такой файл. Должна быть возможность редактировать такой файл настроек на компьютере.
Должна быть указана высокая производительность по мощности дозы (при отключенном высоком напряжении).
Должны быть указаны характеристики сигнализации…
Файлы данных
Файлы спектров с результатами идентификации
Файлы MCS с прикрепленным спектрами
Файлы сигнализации грубого измерения гамма-радиации в режиме СПРД (запись журнала по как минимум 1000
срабатываниям - временная отметка, максимальная скорость счета)
Возможность интегрировать в файлы данных информацию с GPS и цифровые изображения.
Желательный формат файла - XML в соответствии со стандартом ANSI, или хотя бы файлы BSA (*.spe).
